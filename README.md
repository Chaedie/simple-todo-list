# 원티드 프리온보딩 프론트엔드 - 선발 과제

해당 저장소는 "원티드 프리온보딩 프론트엔드 코스" 선발 과제 제출용 저장소입니다.

## 프로젝트 실행 법

```
npm install
npm start
```

## 주요 기능
1) 로그인 / 회원가입
2) 할 일 등록 / 조회 / 수정 / 삭제
